# 更新记录

## 1.1.10 - 2026-07-18

- 主智能体每 80 轮自动保存检查点并继续执行，普通长任务不再要求用户反复点击“继续任务”；只有达到 320 轮总安全上限或连续 32 轮无进展时才暂停。
- 子智能体增加工具参数容错、错误自恢复、受控并发和失败重试，并返回带状态与尝试次数的结构化结果。

## 1.1.9 - 2026-07-18

- 更新扩展市场图标为幽蓝火焰代码灵体，并针对 32px、128px 小尺寸优化轮廓和对比度。
- Activity Bar 单色图标同步采用火焰面具与代码眼睛造型，统一扩展视觉识别。

## 1.1.8 - 2026-07-18

- 将附件入口改为醒目的“上传文件”按钮，并在按钮上显示待发送附件数量。
- 修复文档附件标签渲染时变量名覆盖浏览器 `document` 对象的问题。
- 全面更新中英文 README，统一独立设置页、API Key、智谱端点和附件说明。

## 1.1.7 - 2026-07-18

- 增加独立的“灵境设置”页面，并在侧边栏标题栏提供齿轮入口。
- 模型、接口和密钥集中管理；密钥页只展示配置状态，原文始终保存在 SecretStorage。
- 设置页支持安全更新、删除密钥和使用当前表单配置测试连接。

## 1.1.6 - 2026-07-18

- 在侧边栏输入区工具栏增加“密钥”入口，直接通过密码输入框安全覆盖当前 API Key。
- 增加 `baseUrl` 格式校验，阻止把 API Key 误填为接口地址并给出明确修正提示。

## 1.1.5 - 2026-07-18

- 修复服务商选择为 Z.AI、模型选择 GLM 时仍错误请求智谱国内接口的问题。
- 服务商现在只负责接口地址，模型预设只负责模型名称，仅显式填写的 `baseUrl` 可以覆盖服务商地址。

## 1.1.4 - 2026-07-18

- 将 API Key 的保存、读取和旧配置迁移完整恢复为 1.0 的实现，移除服务商分槽与密钥格式改写。
- 用户粘贴的 Key 仅清理首尾空白，随后直接写入唯一 SecretStorage 记录并用于后续请求。

## 1.1.3 - 2026-07-18

- API Key 恢复为单一当前密钥：每次粘贴都会覆盖旧值，并从下一次请求立即生效。
- 继续使用 VS Code SecretStorage 安全保存，不再按服务商维护多份隐形密钥状态。
- 升级时自动合并并清理 1.1.x 的服务商分槽密钥，避免设置值与实际请求值不一致。

## 1.1.2 - 2026-07-18

- 修复从 1.0 升级后，旧版 Settings 密钥可能未迁移到当前服务商安全存储的问题。
- 迁移时优先采用用户最后一次保存的旧版配置，迁移完成后再清理明文项。

## 1.1.1 - 2026-07-18

- API Key 改为按实际请求服务商分别保存，避免 Qwen、智谱开放平台和智谱 Coding Plan 之间误用密钥。
- 自动迁移旧版共享密钥，并清理用户粘贴内容中的 `Bearer`、引号和换行符。
- 认证失败时显示实际服务商、请求地址和密钥读取状态，便于定位 API Key 与 Base URL 不匹配的问题。

## 1.1.0 - 2026-07-18

- 增加 `apply_patch` 多段文本原子补丁，并通过内容哈希阻止旧上下文覆盖新文件。
- 持久化计划、轮次、文件任务、验证和子智能体状态，支持异常中断后继续执行。
- 增加程序控制的编译、lint、测试流水线和隔离上下文的修改审查。
- 增加项目结构、工作区符号、诊断信息和分层 `AGENTS.md` 指令读取。
- 增加只读、工作区写入、逐次询问三种权限模式及受控后台进程。
- 增加 Git 状态、Diff、分支、提交、推送、Worktree 和 PR 工具。
- 增加 MCP stdio 服务器接入、连接超时、环境变量继承和调用审批。
- 增加最多 3 个只读子智能体并行调查，并在面板展示各子任务状态。
- 长任务执行预算耗尽时改为可恢复的暂停状态，继续任务会获得新的局部预算，不再卡在累计轮次上限。
- 思考过程改为固定最大高度的独立滚动区域，避免长推理挤走最终答复。
- 批量修改默认按 300-500 行读取，同一代码区块尽量合并 5-30 个补丁，并对重复读取、连续小补丁给出运行时效率提醒。

## 1.0.0 - 2026-07-18

- 完成读取、搜索、局部替换、写入和修改后验证的编码闭环。
- 增加复杂任务规划、真实文件任务清单和逐步进度展示。
- 支持图片、DOCX、PPTX、XLSX、PDF、RTF、OpenDocument、EPUB 和 UTF-8 文本附件。
- 支持在输入框中使用 `@` 搜索并引用工作区文件。
- 增加原子写入、修改快照、原生 Diff 和最近一次任务撤销。
- 增加工作区信任检查、真实路径边界检查和受限命令执行。
- API Key 改用 VS Code SecretStorage 保存，并兼容迁移旧版明文设置。
- 增加模型请求超时、瞬时错误重试、连接测试和脱敏诊断日志。
- 增加按工作区恢复的模型上下文、字符预算压缩和长任务防循环机制。
- 增加核心自动化测试，覆盖原子写入、上下文压缩和附件解析。
